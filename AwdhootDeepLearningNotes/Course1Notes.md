# 1. What is neural network
A **Neural Network** is a Machine learning model inspired by the way biological neurons process information. It consists of multiple interconnected nodes(**neurons**) organised into layers.

A Neural Network learns patterns directly from data by adjusting its internal parameters during training.

Neural networks are capable of solving complex problems such as image recognization, speech recognization, and language translation.

# Supervised learning with neural networks
In supervised learning, a neural network is trained using **labeled data**, where each input has a corresponding correct output.The goal is to learn a mapping from inputs to outputs so model can accurately predict the labels of unseen data.

# Binary Classification
Binary Classification is a supervised learning task in which the model predicts one of **two possible outputs**, usually represented as **0** and **1**. where 0 and 1 can be used to denote particular class.

Some examples are :- span vs non-span, fraud vs non-fraud, healthy vs unhealthy, etc.

# Logistic Regression
Logistics Regression is a supervised learning algorithm used for **binary classification and Regression**. It classify on the basis of the probability that an input belongs to a particular class.

It uses **Activation function** to introduce non-linearnity in learning process.

# Gradient Descent
Gradient descent is an optimization algorithm used to train machine learning models by minimizing the **Cost (loss) function** It works by repeatedly updating the model parameters (weights and bias) in the direction where the cost decreases the fastest.
